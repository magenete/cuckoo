Cuckoo
=========

A desktop-oriented virtual machines (based on QEMU) management system written in Shell.


Installation
------------

By network:

    curl -SL https://raw.githubusercontent.com/magenete/cuckoo/master/cuckoo/bin/cuckoo-installer | sh

Local:

    cuckoo-installer -i -v


Running
------------

    cuckoo -h
